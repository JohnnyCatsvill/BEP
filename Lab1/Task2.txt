a) 404 Not Found - Файл не найден, надо добавить файл по указанному адресу
b) 200 OK - Ошибок нет
с) 302 Found - Ресурс отстутствует временно это специальное поведение, либо добавить ресурс, либо перейти напрямую на переадресованный сайт. В файле php_is_a_server_scripting_language.php лежит заголовок который перенаправляет загрузку на сторонний сайт -> Location: http://fi2.php.net/manual/ru/
d) HTTP ERROR 400 - Плохой запрос от клиента, надо сделать другой запрос (hardly -> hard) этот вид запроса не был предусмотрен в php файле, и оно отклонило его вернув ошибку 400.
if (!isset($_GET['how_to_work']) || $_GET['how_to_work'] != 'hard')
{
